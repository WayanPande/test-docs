### UI and Styling

- [Tailwind CSS](https://tailwindcss.com) - A utility-first CSS framework for rapid UI development.
- [Chakra UI](https://chakra-ui.com) - A simple, modular and accessible component library that gives you the building blocks you need to build your React applications.
- [DaisyUI](https://daisyui.com) - Adds component classes to Tailwind CSS for faster UI building.
- [Material-UI](https://mui.com) - React components for faster and easier web development using Material Design.
- [Ant Design](https://ant.design) - A design system with values of nature and clarity for better user experience in enterprise-level products.
- [shadcn.ui](https://ui.shadcn.com) - A UI library based on Tailwind CSS that have different approach from most of UI library out there (Love at first sight ❤️)
